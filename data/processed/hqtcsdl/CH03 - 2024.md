# PL/SQL là gì?

- SQL: Structure Query Language
- SQL chưa đủ mạnh để lập trình
- PL/SQL: Procedural Language extensions for SQL

(IF...THEN.. ELSE ... END IF; LOOP... END LOOP;) + SQL = PL/SQL

- Điểm mạnh của PL/SQL:
    - Tích hợp cấu trúc hướng thủ tục vào SQL
    - Tăng hiệu năng xử lý
    - Module hóa chương trình
    - Khả chuyển
    - Có cơ chế xử lý ngoại lệ


<!-- page 6 -->

# Cách thực thi các lệnh PL/SQL

- **PL/SQL** $\rightarrow$ **PL/SQL Engine**
    - **PL/SQL Block**
        - procedural $\rightarrow$ **Procedural Statement Executor**
        - SQL $\rightarrow$ **SQL Statement Executor**
- **Oracle Database Server**
    - **SQL Statement Executor**

# Giới thiệu về ngôn ngữ PL/SQL

- PL/SQL = SQL chuẩn + thành phần Oracle mở rộng
- Đặc điểm
    - Đơn vị mã lệnh PL/SQL là dạng khối (Begin... End).
    - Mỗi lệnh kết thúc bằng dấu “;”
    - Khối PL/SQL có thể bao gồm nhiều cấu trúc khối PL/SQL khác. Trong đó có thể bao gồm các:
        - Biến & hằng (Variables and constants)
        - Con trỏ (cursor)
        - Xử lý các ngoại lệ (Exceptions)


<!-- page 8 -->

# Cấu trúc khối lệnh PL/SQL

- **DECLARE** – *Optional*
    - Variables, cursors, user – defined exceptions
- **BEGIN** – Mandatory
    - SQL - statements
    - PL/SQL – statements
- **EXCEPTION** – *Optional*
    - Actions to perform when errors occur
- **END;** - Mandatory

Chú thích một dòng bắt đầu bởi - -
Chú thích nhiều dòng được đặt giữa /* và */

# Phạm vi của biến

- **Biến địa phương (Local variables)**
    - These are declared in a inner block and cannot be referenced by outside Blocks.
- **Biến toàn cục (Global variables)**
    - These are declared in a outer block and can be referenced by its itself and by its inner blocks.

```
1> DECLARE
2>  var_num1 number;
3>  var_num2 number;
4> BEGIN
5>  var_num1 := 100;
6>  var_num2 := 200;
7>  DECLARE
8>   var_num3 number;
9>  BEGIN
10>  var_num3 := var_num1 * var_num2;
11> END;
12> END;
```

# Rẽ nhánh

| IF...THEN...ELSE | CASE...WHEN |
| :--- | :--- |
| **IF** *condition* **THEN** | **CASE** *selector* |
| *statements*; | **WHEN** *expression1* **THEN** *result1* |
| [**ELSIF** *condition* **THEN** | **WHEN** *expression2* **THEN** *result2* |
| *statements*;] | . . . |
| [**ELSE** | **WHEN** *expressionN* **THEN** *resultN* |
| *statements*;] | [**ELSE** *resultN+1*] |
| **END IF**; | **END**; |

# Lệnh IF

- **Ví dụ 1:** Nhập n từ bàn phím (nhỏ hơn 1000), đưa ra thông báo n là số chẵn hay lẻ

```sql
declare
    n number(3);
begin
    n:=&n;
    if n mod 2 = 0 then
        dbms_output.put_line(n||' la so chan');
    else
        dbms_output.put_line(n||' la so le');
    end if;
end;
```

# Trigger DML

- **Trigger DML** được tạo trên bảng (table) hoặc khung nhìn (view). Sự kiện kích hoạt của nó gồm các câu lệnh: **INSERT**, **UPDATE**, **DELETE**
- **Trigger DML** đơn được kích hoạt tại các thời điểm:
    - **Before** cấp câu lệnh (statement-level **BEFORE** trigger)
    - **After** cấp câu lệnh (statement-level **AFTER** trigger)
    - **Before** cấp dòng (row-level **BEFORE** trigger)
    - **After** cấp dòng (row-level **AFTER** trigger)
