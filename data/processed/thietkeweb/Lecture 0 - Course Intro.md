## Course description & learning outcomes

This subject provides students with basic and comprehensive knowledge about web designing, including **HTML**, **CSS**, **JavaScript** and modern web design technologies.

After completing the course, students will be able to *identify website layout elements*, to *analyze a web design project requirements*, to *apply appropriate web technologies* to *design a web prototype* and to *deploy the prototype to the Internet*.

Learn Web Design:
- HTML, CSS
- JAVASCRIPT
- JQUERY
- BOOTSTRAP
