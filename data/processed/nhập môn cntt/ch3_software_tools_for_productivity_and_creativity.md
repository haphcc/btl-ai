# Chapter 3: Software: Tools for Productivity & Creativity

## Nội dung chính
- Phần mềm hệ thống (System Software) và vai trò của hệ điều hành (Operating System).
- Device drivers, utility programs và user interface.
- Các hệ điều hành phổ biến.
- Phần mềm ứng dụng (Application Software).
- Data files, program files và cách phần mềm làm việc với dữ liệu.
- Word processing, spreadsheet, database software, office suites và phần mềm chuyên dụng.

## Tổng quan
Chương này giới thiệu phần mềm như công cụ giúp tăng năng suất và hỗ trợ sáng tạo. Nội dung phân biệt phần mềm hệ thống và phần mềm ứng dụng, giải thích vai trò của hệ điều hành, driver, tiện ích, giao diện người dùng và các loại ứng dụng văn phòng hoặc chuyên ngành.

## Unit 3A: System Software: The Power behind the Power

### 3.1 The Operating System

**Khái niệm chính:**
- Hệ điều hành (Operating System - OS) là phần mềm chính điều khiển hoạt động của máy tính.
- OS quản lý phần cứng, phần mềm, file, bộ nhớ, thiết bị nhập xuất và giao diện người dùng.
- Người dùng tương tác với máy tính thông qua OS.

**Giải thích:**
Hệ điều hành là cầu nối giữa người dùng, phần mềm ứng dụng và phần cứng. Khi người dùng mở chương trình, lưu file hoặc in tài liệu, hệ điều hành điều phối tài nguyên để các tác vụ diễn ra đúng.

**Thuật ngữ cần nhớ:**
- `Operating System`: phần mềm quản lý toàn bộ hoạt động cơ bản của máy tính.
- `Booting`: quá trình khởi động máy tính và nạp hệ điều hành.
- `File management`: quản lý file và thư mục.
- `Memory management`: quản lý bộ nhớ khi chương trình chạy.

**Ví dụ / ứng dụng:**
- Windows, macOS, Linux, Android và iOS đều là hệ điều hành.
- Khi cắm USB, OS nhận thiết bị, gán ổ đĩa và cho phép truy cập file.

### 3.2 Other System Software

**Khái niệm chính:**
- Device drivers giúp hệ điều hành giao tiếp với thiết bị phần cứng.
- Utility programs là phần mềm tiện ích hỗ trợ bảo trì, bảo vệ và tối ưu hệ thống.

**Giải thích:**
Không phải mọi phần mềm hệ thống đều là hệ điều hành. Driver giúp máy in, card âm thanh, màn hình hoặc thiết bị khác hoạt động đúng. Utility programs hỗ trợ các tác vụ như nén file, sao lưu, chống virus, dọn ổ đĩa hoặc phục hồi dữ liệu.

| Loại phần mềm | Vai trò |
|---|---|
| `Device driver` | Điều khiển thiết bị phần cứng cụ thể |
| `Utility program` | Bảo trì, bảo vệ hoặc tối ưu hệ thống |
| `Antivirus software` | Phát hiện và xử lý phần mềm độc hại |
| `Backup software` | Sao lưu dữ liệu |

**Thuật ngữ cần nhớ:**
- `Driver`: chương trình giúp OS điều khiển thiết bị.
- `Utility`: tiện ích hệ thống.
- `Backup`: sao lưu dữ liệu để phục hồi khi cần.

**Ví dụ / ứng dụng:**
- Cài driver máy in để in tài liệu.
- Dùng tiện ích chống virus để quét phần mềm độc hại.

### 3.3 Common Features of the User Interface

**Khái niệm chính:**
- Giao diện người dùng (User Interface - UI) là phần người dùng nhìn thấy và tương tác.
- UI có thể là giao diện dòng lệnh hoặc giao diện đồ họa.
- Giao diện đồ họa sử dụng cửa sổ, biểu tượng, menu và con trỏ.

**Giải thích:**
Giao diện giúp người dùng ra lệnh cho máy tính mà không cần hiểu sâu về phần cứng. Giao diện đồ họa (Graphical User Interface - GUI) phổ biến vì trực quan, cho phép nhấp chuột, kéo thả và thao tác với biểu tượng.

**Thuật ngữ cần nhớ:**
- `User Interface`: giao diện người dùng.
- `GUI`: giao diện đồ họa.
- `Icon`: biểu tượng đại diện cho chương trình, file hoặc chức năng.
- `Menu`: danh sách lệnh.

**Ví dụ / ứng dụng:**
- Desktop có biểu tượng, thanh tác vụ, cửa sổ ứng dụng và menu Start là một GUI.

### 3.4 Common Operating Systems

**Khái niệm chính:**
- Có nhiều hệ điều hành cho máy tính cá nhân, máy chủ và thiết bị di động.
- Mỗi OS khác nhau về giao diện, hệ sinh thái phần mềm, phần cứng hỗ trợ và đối tượng người dùng.

**Giải thích:**
Các hệ điều hành phổ biến phục vụ nhu cầu khác nhau. Windows thường gặp trên PC, macOS trên máy Mac, Linux trên server và người dùng kỹ thuật, Android và iOS trên thiết bị di động.

| Hệ điều hành | Nền tảng phổ biến |
|---|---|
| `Windows` | PC, laptop |
| `macOS` | Máy tính Apple |
| `Linux` | Server, máy tính cá nhân, hệ thống nhúng |
| `Android` | Smartphone, tablet |
| `iOS` | iPhone, iPad |

**Thuật ngữ cần nhớ:**
- `Platform`: nền tảng phần cứng/phần mềm để chạy ứng dụng.
- `Mobile OS`: hệ điều hành cho thiết bị di động.
- `Open source`: mã nguồn mở, cho phép xem và sửa mã nguồn theo điều kiện giấy phép.

**Ví dụ / ứng dụng:**
- Ứng dụng viết cho Android không tự động chạy trên iOS nếu không có phiên bản phù hợp.

## Unit 3B: Application Software

### 3.5 Application Software

**Khái niệm chính:**
- Phần mềm ứng dụng (Application Software) giúp người dùng thực hiện công việc cụ thể.
- Ứng dụng có thể dùng cho năng suất, sáng tạo, truyền thông, quản lý, giải trí hoặc chuyên ngành.

**Giải thích:**
Nếu hệ điều hành giúp máy tính hoạt động, thì phần mềm ứng dụng giúp người dùng hoàn thành công việc. Ví dụ soạn văn bản, tính toán bảng biểu, quản lý dữ liệu, chỉnh sửa ảnh, thiết kế, thuyết trình hoặc duyệt web.

**Thuật ngữ cần nhớ:**
- `Application software`: phần mềm phục vụ tác vụ cụ thể của người dùng.
- `Productivity software`: phần mềm tăng năng suất.
- `Specialty software`: phần mềm chuyên dụng cho lĩnh vực cụ thể.

**Ví dụ / ứng dụng:**
- Word processor để viết báo cáo.
- Spreadsheet để tính toán ngân sách.
- Database software để quản lý danh sách khách hàng.

### 3.6 Data Files & Program Files

**Khái niệm chính:**
- Program files chứa mã chương trình.
- Data files chứa dữ liệu do người dùng tạo hoặc chương trình sử dụng.
- Phần mềm ứng dụng mở, xử lý và lưu dữ liệu trong file.

**Giải thích:**
Khi cài phần mềm, máy tính có các file chương trình để chạy ứng dụng. Khi người dùng tạo tài liệu, bảng tính hoặc ảnh, kết quả được lưu thành file dữ liệu. Phân biệt hai loại file giúp quản lý và sao lưu đúng.

**Thuật ngữ cần nhớ:**
- `Program file`: file chứa mã chương trình.
- `Data file`: file chứa dữ liệu.
- `File extension`: phần mở rộng file giúp nhận diện loại file.

**Ví dụ / ứng dụng:**
- File `.docx` là dữ liệu văn bản; file thực thi của phần mềm là program file.

### 3.7 Word Processing Software

**Khái niệm chính:**
- Word processing software dùng để tạo, chỉnh sửa, định dạng và in tài liệu văn bản.
- Các chức năng thường có gồm kiểm tra chính tả, định dạng đoạn, chèn hình, bảng và in ấn.

**Giải thích:**
Phần mềm xử lý văn bản giúp người dùng tạo tài liệu chuyên nghiệp hơn so với gõ văn bản thô. Người dùng có thể thay đổi font, căn lề, đánh số trang, chèn bảng, hình ảnh và kiểm tra lỗi.

**Thuật ngữ cần nhớ:**
- `Word processor`: phần mềm xử lý văn bản.
- `Formatting`: định dạng văn bản.
- `Template`: mẫu tài liệu.

**Ví dụ / ứng dụng:**
- Soạn báo cáo, đơn xin việc, bài luận hoặc thư.

### 3.8 Spreadsheet Programs

**Khái niệm chính:**
- Spreadsheet programs dùng để tổ chức dữ liệu theo hàng và cột.
- Bảng tính hỗ trợ công thức, hàm, biểu đồ và phân tích số liệu.

**Giải thích:**
Bảng tính phù hợp với dữ liệu số như ngân sách, điểm số, doanh thu hoặc kế hoạch. Người dùng nhập dữ liệu vào ô, dùng công thức để tính toán và tạo biểu đồ để minh họa.

**Thuật ngữ cần nhớ:**
- `Spreadsheet`: bảng tính.
- `Cell`: ô trong bảng tính.
- `Formula`: công thức.
- `Chart`: biểu đồ.

**Ví dụ / ứng dụng:**
- Tính tổng chi tiêu hằng tháng.
- Lập bảng điểm và tính trung bình.

### 3.9 Database Software

**Khái niệm chính:**
- Database software dùng để lưu trữ, tổ chức, truy vấn và quản lý dữ liệu có cấu trúc.
- Cơ sở dữ liệu (Database) thường gồm bảng, bản ghi và trường.

**Giải thích:**
Khi dữ liệu lớn hoặc cần tìm kiếm, lọc, cập nhật thường xuyên, phần mềm cơ sở dữ liệu hiệu quả hơn bảng tính. Cơ sở dữ liệu giúp quản lý thông tin khách hàng, hàng hóa, đơn hàng hoặc nhân sự.

**Thuật ngữ cần nhớ:**
- `Database`: cơ sở dữ liệu.
- `Record`: bản ghi, một dòng dữ liệu về một đối tượng.
- `Field`: trường, một thuộc tính dữ liệu.
- `Query`: truy vấn dữ liệu.

**Ví dụ / ứng dụng:**
- Danh sách sinh viên gồm các trường mã sinh viên, họ tên, lớp, email.

### 3.10 Software Suites & 3.11 Specialty Application Software

**Khái niệm chính:**
- Office suite là bộ phần mềm văn phòng tích hợp nhiều ứng dụng.
- Specialty application software là phần mềm chuyên dụng cho nghề nghiệp hoặc mục đích cụ thể.

**Giải thích:**
Bộ phần mềm văn phòng thường gồm xử lý văn bản, bảng tính, trình chiếu, email và cơ sở dữ liệu. Phần mềm chuyên dụng có thể phục vụ thiết kế đồ họa, kế toán, quản lý dự án, y tế, giáo dục hoặc kỹ thuật.

**Thuật ngữ cần nhớ:**
- `Office suite`: bộ phần mềm văn phòng.
- `Presentation software`: phần mềm trình chiếu.
- `Specialty application`: ứng dụng chuyên dụng.

**Ví dụ / ứng dụng:**
- Bộ office dùng cho tài liệu, bảng tính và thuyết trình.
- Phần mềm kế toán dùng cho doanh nghiệp.
